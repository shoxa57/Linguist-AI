# Linguist AI: Neural Translation Core 🌐

A high-performance translation engine built with Python and integrated into a futuristic web interface. This project demonstrates the bridge between Python backend logic and modern frontend design, featuring real-time neural processing simulation.

## 🚀 Key Features
- **Neural Processing:** Leverages advanced NLP models for accurate text translation.
- **Dynamic Interface:** Cyberpunk-inspired UI with real-time status indicators.
- **Language Detection:** Automatically identifies source language for a seamless user experience.
- **Async Architecture:** Non-blocking translation requests using JavaScript asynchronous calls.

## 🛠 Technology Stack
- **Language:** Python 3.x
- **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (ES6+)
- **Libraries:** Deep-Translator (Python), FontAwesome, Google Fonts
- **Integration:** REST API simulation for portfolio interaction

## 📂 Project Structure
- `translator_core.py`: The main Python logic for translation.
- `index.html`: The HTML structure for the project card.
- `script.js`: Integration logic to connect the UI with the translation engine.
